# ARCHIVIRT Framework

**A**utomated **R**eproducible **C**yber **H**yberd **I**nfrastructure for **VIRT**ual IDS/IPS Testing Labs

## Описание

ARCHIVIRT — это фреймворк с открытым исходным кодом для полной автоматизации жизненного цикла виртуальных лабораторий, предназначенных для тестирования и сравнения систем обнаружения и предотвращения вторжений (IDS/IPS).

## Ключевые особенности

- **Infrastructure as Code**: Полное описание инфраструктуры в Terraform и Ansible
- **Модульная архитектура**: Легкая замена компонентов (цели, IDS, инструменты атак)
- **Автоматизированный конвейер тестирования**: От развертывания до генерации отчетов
- **Воспроизводимость**: Идентичные среды из кода
- **Расширяемая библиотека сценариев**: YAML-формат для описания тестовых сценариев
